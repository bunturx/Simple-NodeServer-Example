# Simple NodeServer Example for the students group

## USE COMMANDS
- npm i
- npm start


Server run in port 3000

## ENDPOINTS
```
GET
- /photos
- /photos/:indexPhoto

POST
- /photos

PUT
- /photos/:id

DELETE
- /photos/:id
```
